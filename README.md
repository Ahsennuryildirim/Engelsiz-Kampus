# Engelsiz Kampüs

Engelli bireylerin kampüs ve şehir içindeki erişilebilir alanları görüntüleyebilmesi amacıyla geliştirilmiş web projesidir.

## Özellikler

- Kampüs erişilebilirlik sistemi
- Şehir erişilebilirlik sistemi
- Hastane randevu sistemi
- Favori mekanlar
- Arama ve filtreleme
- Tema desteği
- İstatistikler

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript
- Bootstrap
- LocalStorage
